# ⭐ Dashboard Features

The Retail Sales & Profitability Dashboard includes a set of interactive, analytical, and design‑focused features that help users explore sales performance, profitability, and return behavior across multiple business dimensions.

---

## 🔍 High‑Level Features
- Interactive slicers for Category, Region, Segment, and Date  
- Dynamic visuals that update instantly based on user selections  
- Integrated return analysis merged directly into the Orders dataset  
- Clean, modern layout optimized for executive decision‑making  
- Automated data refresh using Power Query  

---

## 📊 KPI Features
- Total Sales  
- Total Profit  
- Profit Margin %  
- Return Rate %  
- YoY performance indicators  

---

## 📈 Visual Features
- Monthly sales trend line chart  
- Profit by region bar chart  
- Category and sub‑category performance breakdown  
- Return rate comparison visual  
- Top‑performing products table  

---

## 🛠️ Data Features
- Power Query transformations for data cleaning  
- Merged Orders + Returns dataset  
- Profit Margin calculated column  
- Return Flag logic  
- Standardized data types for accurate analysis  

---

## 🎨 User Experience Features
- Clean white KPI cards with icons  
- Intuitive slicer panel for easy filtering  
- Balanced three‑column layout  
- Color‑coded profit indicators  
- Clear typography and spacing for readability
