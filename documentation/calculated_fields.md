📊 5. Calculated Fields
| Calculated Field | Formula / Logic | Description |
| --- | --- | --- |
| **Profit Margin** | ``Profit ``/ ``Sales`` | Measures profitability relative to revenue |
| **Discount Impact** | ``Sales ``* ``Discount`` | Shows revenue lost due to discounting |
| **Sales After Discount** | ``Sales ``- ``(Sales ``* ``Discount)`` | Adjusted revenue after discount |
| **Return Rate** | ``Returned ``Orders ``/ ``Total ``Orders`` | Measures product or region return frequency |
| **Avg Order Value (AOV)** | ``Total ``Sales ``/ ``Number ``of ``Orders`` | Average revenue per order |
| **YoY Sales Growth** | ``(Current ``Year ``Sales ``- ``Prior ``Year ``Sales) ``/ ``Prior ``Year ``Sales`` | Measures annual sales growth |
